# Comsol 

## Description of the Comsol Command

```-nn```
Number of nodes

```-np```
Number of cores per nodes

```-f```
The file that saves the names of nodes

```-inputfile```
The input file 

```-outputfile```
The output file

```-batchlog```
The log file. You could use ```tail -f test.log``` to watch the running process.
